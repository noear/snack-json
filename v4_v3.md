## Json 定制

* 全局 CodecLib（一般框架内部使用）

```
CodecLib.addFactory(.)
CodecLib.addDecoder(.)
CodecLib.addEncoder(.)
```

* 局部 Options

```
Options.of().addFactory(.).addDecoder(.).addEncoder(.).addFeature(.)
```
