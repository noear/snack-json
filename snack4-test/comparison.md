

|          | `IETF JSONPath (RFC 9535)`        | `jayway.jsonpath`        |
|----------|-----------------------------------|--------------------------|
| 过滤器的过滤目标 | array:子项；object:值项；value：自己       | array:后代；object,value：自己 |
| `..`     | 联合才可为选择器，如：`..name`，`..*`，`..[x]` | 独立是个选择器                  |
|          | `$..[?@.b.c == 1]`                | 对应 `$..b[?(@.c == 1)]`   |
